  <name>Task 1: Define blocklist in canonical location</name>
  <files>scripts/config.py</files>
  <action>
Define blocklist in scripts/config.py (canonical location):
1. Create BLOCKLIST set with:
   - `rm -rf /` (root deletion)
   - `rm -rf /*` (broad deletion)
   - `mkfs` (filesystem format)
   - `mkfs.*` (any mkfs variant)
   - `dd if=/dev/zero` (disk wipe pattern)
   - `:(){ :|:& };:` (fork bomb)
   - `wget | sh` (remote code download+exec)
   - `curl | sh` (remote code download+exec)
2. Make BLOCKLIST immutable (no override in config)
3. Document: blocklist is security-critical, never bypassable
4. Consider adding more patterns based on security research

This is REQUIRED per REQUIREMENTS.md: "non-bypassable blocklist"
</action>
