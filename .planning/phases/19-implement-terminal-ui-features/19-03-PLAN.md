<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| model/tool output -> renderer | Untrusted streamed text becomes rich formatted blocks and previews. |
| tool arguments/results -> sidebar | Tool metadata shown in the timeline may contain long or malformed payloads. |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-19-07 | I | sidebar context preview | mitigate | Show repo-relative display names and bounded first-40-line previews only; avoid dumping full raw file contents or absolute home paths directly into the sidebar. |
| T-19-08 | D | markdown/diff renderer | mitigate | Use line-based parsing with bounded preview lengths and preserved UTF-8 slices so malformed model output cannot trigger panics or unbounded layout growth. |
| T-19-09 | T | tool timeline state | mitigate | Key timeline items by structured tool `id` and update status/duration on explicit `ToolResult` events so late or duplicate events cannot overwrite unrelated entries. |
</threat_model>
