<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| terminal user -> palette/input | Untrusted keystrokes and edited temp-file content become commands or prompt text. |
| filesystem -> theme loader | User config files under `~/.config/helix-agent` are untrusted input to the renderer. |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-19-04 | T | slash command palette | mitigate | Limit immediate execution to the explicitly enumerated zero-argument commands and insert all argument-bearing commands as literal text so typed arguments stay user-visible before dispatch. |
| T-19-05 | I | theme config loader | mitigate | Parse only the allowed `[colors]` keys and `#RRGGBB` values from the user config path; ignore unknown keys and malformed lines instead of propagating arbitrary file content into styles. |
| T-19-06 | D | external editor flow | mitigate | Use temp-file round-trip error handling plus `StatusBanner` reporting so editor launch/read failures do not crash the TUI or corrupt the in-memory input buffer. |
</threat_model>
